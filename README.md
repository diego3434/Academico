Este proyecto fue desarrollado en Java aplicando Programación Orientada a Objetos mediante el uso de interfaces.
El sistema permite imprimir diferentes tipos de documentos académicos como certificados, actas de notas y horarios académicos.

Se implementó la interfaz Imprimible, la cual es utilizada por las clases:

Certificado
ActaNotas
HorarioAcademico

Cada clase sobrescribe el método imprimir() usando @Override, mostrando información diferente según el tipo de documento.

Además, el programa utiliza Scanner para ingresar datos desde consola y demostrar el funcionamiento del sistema.

Conceptos aplicados:

Interfaces
Polimorfismo
Encapsulamiento
Sobrescritura de métodos
Programación Orientada a Objetos en Java
